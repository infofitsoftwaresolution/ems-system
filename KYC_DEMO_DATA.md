
## Document Uploads

For the file uploads, you'll need to upload actual image files:

1. **Employee Photo** (Required)

   - Format: JPEG, JPG, PNG, or WebP
   - Max size: 5MB
   - Use any passport-size photo or selfie

2. **PAN Card Photo** (Required)

   - Format: JPEG, JPG, PNG, or WebP
   - Max size: 5MB
   - Use any sample PAN card image

3. **Aadhar Card Photo** (Required)
   - Format: JPEG, JPG, PNG, or WebP
   - Max size: 5MB
   - Use any sample Aadhar card image
